# Intal
A Big Integer libarry 

Following functions are provided by this library

1. add
2. compare
3. diff
4. multiply
5. div
6. mod
7. pow
8. gcd
9. max
10. min
11. sort
12. search
13. binsearch
14. bincoeff
15. coin row problem
16. fibonacii
17. factorial

The project uses cunit for unit testing. Refer http://cunit.sourceforge.net/ 

Commands:
 gcc test.c implementation.c -lcunit
